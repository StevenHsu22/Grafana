<h2 id="use">用途</h2>

可將外部內容用 GET、POST 或 iframe 加載到儀表板(支援 TEXT、JSON、Bolb、Images)

<h2 id="install">安裝方式、有無支援 ElasticSearch</h2>

搜尋 Grafana Plugins 中的 AJAX 並點擊 INSTALL 或打以下指令

    grafana-cli plugins install ryantxu-ajax-panel
