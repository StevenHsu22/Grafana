<h2 id="use">用途</h2>

與 Grafana Singlestat 幾乎完全相同，只是它允許多個序列或查詢

<h2 id="install">安裝方式、有無支援 ElasticSearch</h2>

搜尋 Grafana Plugins 中的 ACE.SVG 並點擊 INSTALL 或打以下指令

    grafana-cli plugins install farski-blendstat-panel
