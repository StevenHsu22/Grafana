<h2 id="use">用途</h2>

能同時監控多個設備的數據

<h2 id="install">安裝方式、有無支援 ElasticSearch</h2>

搜尋 Grafana Plugins 中的 Boom Table 並點擊 INSTALL 或打以下指令

    grafana-cli plugins install yesoreyeram-boomtable-panel
