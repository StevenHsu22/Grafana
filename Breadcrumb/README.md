<h2 id="use">用途</h2>

跟蹤使用者訪問過的儀錶板，並將其顯示為痕跡導航

<h2 id="install">安裝方式、有無支援 ElasticSearch</h2>

搜尋 Grafana Plugins 中的 ACE.SVG 並點擊 INSTALL 或打以下指令

    grafana-cli plugins install digiapulssi-breadcrumb-panel
