<h2 id="use">用途</h2>

可創建按鈕並為它們定義操作。如設定刪除 InfluxDB 數據庫的查詢字段(JSON語法)

<h2 id="install">安裝方式、有無支援 ElasticSearch</h2>

搜尋 Grafana Plugins 中的 Button 並點擊 INSTALL 或打以下指令

    grafana-cli plugins install yesoreyeram-boomtheme-panel
