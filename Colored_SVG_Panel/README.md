<h2 id="use">用途</h2>

可自定義的閾值將數據顯示為彩色 svg

<h2 id="install">安裝方式、有無支援 ElasticSearch</h2>

搜尋 Grafana Plugins 中的 Colored SVG Panel 並點擊 INSTALL 或打以下指令

    grafana-cli plugins install snuids-svg-panel
