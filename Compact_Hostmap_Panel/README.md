可以一目了然的呈現主機的系統指標

<h2 id="install">安裝方式、有無支援 ElasticSearch</h2>

搜尋 Grafana Plugins 中的 Compact Hostmap Panel 並點擊 INSTALL 或打以下指令

    grafana-cli plugins install zestairlove-compacthostmap-panel

*也支援 promethues*
