<h2 id="use">用途</h2>

基於 D3 的儀表板。可調閥值、針的動畫速度、度數等

<h2 id="install">安裝方式、有無支援 ElasticSearch</h2>

搜尋 Grafana Plugins 中的 D3 Gauge 並點擊 INSTALL 或打以下指令

    grafana-cli plugins install briangann-gauge-panel
