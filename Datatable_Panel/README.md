<h2 id="use">用途</h2>

與內置 Grafana Table Panel 功能相同，可依使用者需求將閥值整行或單格變色處理

<h2 id="install">安裝方式、有無支援 ElasticSearch</h2>

搜尋 Grafana Plugins 中的 Datatable Panel 並點擊 INSTALL 或打以下指令

    grafana-cli plugins install briangann-datatable-panel
