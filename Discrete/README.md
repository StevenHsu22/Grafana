<h2 id="use">用途</h2>

此面板可在水平圖中顯示離散值，且可清楚地顯示狀態轉換

<h2 id="install">安裝方式、有無支援 ElasticSearch</h2>

搜尋 Grafana Plugins 中的 Discrete 並點擊 INSTALL 或打以下指令

    grafana-cli plugins install natel-discrete-panel
