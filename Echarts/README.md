<h2 id="use">用途</h2>

可使用 react 進行編碼，客製化程度高

<h2 id="install">安裝方式、有無支援 ElasticSearch</h2>

搜尋 Grafana Plugins 中的 Echarts 並點擊 INSTALL 或打以下指令

    grafana-cli plugins install bilibala-echarts-panel
