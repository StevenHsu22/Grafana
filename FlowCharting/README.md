<h2 id="use">用途</h2>

繪製大範圍流程圖，為其提供實時數據並在流程圖中定義數據如何影響視覺反饋

<h2 id="install">安裝方式、有無支援 ElasticSearch</h2>

搜尋 Grafana Plugins 中的 FlowCharting 並點擊 INSTALL 或打以下指令

    grafana-cli plugins install agenty-flowcharting-panel
