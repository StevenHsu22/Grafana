<h2 id="use">用途</h2>

可在迴圈中對地理要素進行動畫處理，可以將多邊形、點或線屬性連結到通過 Grafana 獲取的動態數據

<h2 id="install">安裝方式、有無支援 ElasticSearch</h2>

搜尋 Grafana Plugins 中的 GeoLoop 並點擊 INSTALL 或打以下指令

    grafana-cli plugins install citilogics-geoloop-panel

*不支援 ES*
