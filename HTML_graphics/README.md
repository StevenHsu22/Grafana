<h2 id="use">用途</h2>

透過自行撰寫 HTML 或 SVG 呈現圖

<h2 id="install">安裝方式、有無支援 ElasticSearch</h2>

搜尋 Grafana Plugins 中的 HTML graphics 並點擊 INSTALL 或打以下指令

    grafana-cli plugins install gapit-htmlgraphics-panel
