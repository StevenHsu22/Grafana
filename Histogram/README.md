<h2 id="install">安裝方式、有無支援 ElasticSearch</h2>

搜尋 Grafana Plugins 中的 Histogram (mtanda) 並點擊 INSTALL 或打以下指令

    grafana-cli plugins install mtanda-histogram-panel

*並未支援 ElasticSearch*
