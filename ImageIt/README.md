<h2 id="install">安裝方式、有無支援 ElasticSearch</h2>

搜尋 Grafana Plugins 中的 ImageIt 並點擊 INSTALL 或打以下指令

    grafana-cli plugins install pierosavi-imageit-panel

*不支援 Grafana 9*
