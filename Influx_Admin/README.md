<h2 id="use">用途</h2>

複製 InfluxDB 的網頁管理介面，僅支援 InfluxDB 數據源

<h2 id="install">安裝方式、有無支援 ElasticSearch</h2>

搜尋 Grafana Plugins 中的 Influx Admin 並點擊 INSTALL 或打以下指令

    grafana-cli plugins install natel-influx-admin-panel

*不支援 ES*
