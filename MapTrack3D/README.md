<h2 id="use">用途</h2>

在 3D 地球上將 GPS 點視覺化

<h2 id="install">安裝方式、有無支援 ElasticSearch</h2>

搜尋 Grafana Plugins 中的 MapTrack3D 並點擊 INSTALL 或打以下指令

    grafana-cli plugins install flaminggoat-maptrack3d-panel

*不支援 ES，只支援 MySQL / MariaDB*
