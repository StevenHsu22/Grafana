<h2 id="use">用途</h2>

能將時間序的資料放入製作帕累托圖表

<h2 id="install">安裝方式、有無支援 ElasticSearch</h2>

搜尋 Grafana Plugins 中的 Pareto Chart 並點擊 INSTALL 或打以下指令

    grafana-cli plugins install isaozler-paretochart-panel
