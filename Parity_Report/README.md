<h2 id="use">用途</h2>

可比較各資料差異率，並設定閥值

<h2 id="install">安裝方式、有無支援 ElasticSearch</h2>

搜尋 Grafana Plugins 中的 Parity Report 並點擊 INSTALL 或打以下指令

    grafana-cli plugins install zuburqan-parity-report-panel
