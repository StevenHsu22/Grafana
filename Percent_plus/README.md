<h2 id="use">用途</h2>

只需計算並顯示給定兩個指標的百分比

<h2 id="install">安裝方式、有無支援 ElasticSearch</h2>

搜尋 Grafana Plugins 中的 Percent+ 並點擊 INSTALL 或打以下指令

    grafana-cli plugins install jeanbaptistewatenberg-percent-panel
