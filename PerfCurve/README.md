<h2 id="use">用途</h2>

用於在面板上繪製旋轉機器的操作點，並附有曲線功能

<h2 id="install">安裝方式、有無支援 ElasticSearch</h2>

搜尋 Grafana Plugins 中的 PerfCurve 並點擊 INSTALL 或打以下指令

    grafana-cli plugins install sskgo-perfcurve-panel
