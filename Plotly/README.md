<h2 id="use">用途</h2>

使用 plot.ly javascript 框架呈現指標

<h2 id="install">安裝方式、有無支援 ElasticSearch</h2>

搜尋 Grafana Plugins 中的 Plotly 並點擊 INSTALL 或打以下指令

    grafana-cli plugins install natel-plotly-panel

*適用於 grafana 4、5 和 6*
