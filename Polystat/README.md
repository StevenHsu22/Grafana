<h2 id="use">用途</h2>

為每個指標創建一個六邊形圖標，能夠將指標分組為複合指標，並為各指標設上閥值

<h2 id="install">安裝方式、有無支援 ElasticSearch</h2>

搜尋 Grafana Plugins 中的 Polystat 並點擊 INSTALL 或打以下指令

    grafana-cli plugins install grafana-polystat-panel
