<h2 id="use">用途</h2>

顯示一個板中類似進度的項目清單

<h2 id="install">安裝方式、有無支援 ElasticSearch</h2>

搜尋 Grafana Plugins 中的 Progress List 並點擊 INSTALL 或打以下指令

    grafana-cli plugins install corpglory-progresslist-panel
