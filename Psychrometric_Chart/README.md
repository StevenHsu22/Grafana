<h2 id="use">用途</h2>

用於繪製空氣-蒸汽混合物的各種熱力學特性

<h2 id="install">安裝方式、有無支援 ElasticSearch</h2>

搜尋 Grafana Plugins 中的 Psychrometric Chart 並點擊 INSTALL 或打以下指令

    grafana-cli plugins install ventura-psychrometric-panel
