<h2 id="use">用途</h2>

可將時間範圍調整為一個特定的班次或一系列班次以顯示其他圖表的結果

<h2 id="install">安裝方式、有無支援 ElasticSearch</h2>

搜尋 Grafana Plugins 中的 Shift Selector 並點擊 INSTALL 或打以下指令

    grafana-cli plugins install isaozler-shiftselector-panel
