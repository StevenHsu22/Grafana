<h2 id="use">用途</h2>

將圖片進一步轉換為3D圖形，將數據顯著的區塊顯示隆起狀

<h2 id="install">安裝方式、有無支援 ElasticSearch</h2>

搜尋 Grafana Plugins 中的 Sierra Plot 並點擊 INSTALL 或打以下指令

    grafana-cli plugins install boazreicher-sierraplot-panel
