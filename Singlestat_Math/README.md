<h2 id="use">用途</h2>

將 Singlestat 升級，可支援跨序列的數學函數

<h2 id="install">安裝方式、有無支援 ElasticSearch</h2>

搜尋 Grafana Plugins 中的 Singlestat Math 並點擊 INSTALL 或打以下指令

    grafana-cli plugins install blackmirror1-singlestat-math-panel
