<h2 id="install">安裝方式、有無支援 ElasticSearch</h2>

搜尋 Grafana Plugins 中的 Statusmap 並點擊 INSTALL 或打以下指令

    grafana-cli plugins install flant-statusmap-panel

*支援 Prometheus / Mysql / Grafana 7.x, 8.x*
