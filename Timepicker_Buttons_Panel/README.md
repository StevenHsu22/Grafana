<h2 id="use">用途</h2>

用於設置 Grafana 儀表板的時間範圍，可自訂時間區間

<h2 id="install">安裝方式、有無支援 ElasticSearch</h2>

搜尋 Grafana Plugins 中的 Timepicker Buttons Panel 並點擊 INSTALL 或打以下指令

    grafana-cli plugins install williamvenner-timepickerbuttons-panel
