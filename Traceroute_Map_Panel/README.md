<h2 id="use">用途</h2>

可視化地圖中的 IP位址端點

<h2 id="install">安裝方式、有無支援 ElasticSearch</h2>

搜尋 Grafana Plugins 中的 Traceroute Map Panel 並點擊 INSTALL 或打以下指令

    grafana-cli plugins install gowee-traceroutemap-panel
