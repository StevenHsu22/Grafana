<h2 id="use">用途</h2>

將 GPS 點視覺化為互動式地圖上的一條線

<h2 id="install">安裝方式、有無支援 ElasticSearch</h2>

搜尋 Grafana Plugins 中的 TrackMap 並點擊 INSTALL 或打以下指令

    grafana-cli plugins install pr0ps-trackmap-panel

*不支援 ES，支援 MySQL / MariaDB*
