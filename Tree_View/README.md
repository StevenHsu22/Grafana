<h2 id="install">安裝方式、有無支援 ElasticSearch</h2>

搜尋 Grafana Plugins 中的 Tree View 並點擊 INSTALL 或打以下指令

    grafana-cli plugins install pgillich-tree-panel

*可支援 ES，也支援 JSON API 數據源*
