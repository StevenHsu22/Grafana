<h2 id="use">用途</h2>

用於將時間序列數據顯示為距離函數

<h2 id="install">安裝方式、有無支援 ElasticSearch</h2>

搜尋 Grafana Plugins 中的 Untimely 並點擊 INSTALL 或打以下指令

    grafana-cli plugins install factry-untimely-panel
