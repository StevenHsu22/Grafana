<h2 id="use">用途</h2>

用於顯示來自 URL、YouTube ID 或 iFrame 的影片的面板

<h2 id="install">安裝方式、有無支援 ElasticSearch</h2>

搜尋 Grafana Plugins 中的 Video 並點擊 INSTALL 或打以下指令

    grafana-cli plugins install innius-video-panel
