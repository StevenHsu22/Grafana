<h2 id="use">用途</h2>

可將網站連結設置到儀表板

<h2 id="install">安裝方式、有無支援 ElasticSearch</h2>

搜尋 Grafana Plugins 中的 Website Navigation 並點擊 INSTALL 或打以下指令

    grafana-cli plugins install lework-lenav-panel
