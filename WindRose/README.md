<h2 id="use">用途</h2>

用來簡單描述某一地區風向風速的分布

<h2 id="install">安裝方式、有無支援 ElasticSearch</h2>

搜尋 Grafana Plugins 中的 WindRose 並點擊 INSTALL 或打以下指令

    grafana-cli plugins install fatcloud-windrose-panel
