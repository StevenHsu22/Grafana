透過URL連結來顯示圖像，可搭配即時指標顯示值

<h2 id="install">安裝方式、有無支援 ElasticSearch</h2>

搜尋 Grafana Plugins 中的 ePict Panel 並點擊 INSTALL 或打以下指令

    grafana-cli plugins install larona-epict-panel
